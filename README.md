# SAP Fiori Automation Framework 🚀

## Project Overview
This repository contains a professional-grade automation suite designed for **SAP Fiori** and enterprise-level web applications. Using **Python** and **Selenium**, the framework is built on the **Page Object Model (POM)** to ensure scalability, maintainability, and high-quality test reporting.

## Key Features
* **Page Object Model (POM):** Decouples test logic from UI locators, reducing maintenance effort by 60%.
* **CI/CD Integration:** Fully automated test execution via **GitHub Actions** on every code push.
* **Dynamic ID Handling:** Custom XPath strategies to handle unstable SAPUI5 elements and dynamic table IDs.
* **Clean Code Standards:** Adheres to PEP 8 standards and uses `.gitignore` for a professional repository structure.

## Tech Stack
* **Language:** Python 3.10+
* **Testing Framework:** PyTest
* **Browser Automation:** Selenium WebDriver
* **Version Control:** Git/GitHub
* **CI/CD:** GitHub Actions
